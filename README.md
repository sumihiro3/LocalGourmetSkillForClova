# LocalGourmetSkillForClova

Clova Extension Kit SDK for Python を使ったスキルのサンプルです。

## どんなスキル？
各都道府県にあるご当地グルメ情報を教えてくれるスキルです。
このスキルでできることは次の2種類です。

* 都道府県名からご当地グルメを調べる
    * 例：北海道のご当地グルメは？
* 個別のご当地グルメの詳細を調べる
    * 例：ザンギのことを教えて
